<h4 align="left">Projeto Eagles - ViagensFX:</h4><p align="left">Sistema desktop desenvolvido em Java + JavaFX + MySQL para gerenciamento de registros relacionados a viagens, envolvendo prisioneiros, passaportes e rodovias. O sistema permite realizar operações completas de cadastro, edição, listagem e exclusão de dados, com interface gráfica intuitiva.</p>
